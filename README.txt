#################################################################
#This project provides example of simple android calculator test#
#################################################################

Precondition:
1) environment should be set up correctly:
- whole android SDK should be uploaded and added to IDE and ANDROID_HOME declared as well
- install .apk on device or emulator ('intelhaxm' for emulator should be installed)
- appium server should be installed (will be started from code automatically)

1) 'src/main/resources/android.properties' should be configured accordingly:
- device.name should be specified
- other values should be replaced according to your needs
